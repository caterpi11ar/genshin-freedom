# genshin-freedom 云原神每日领取月卡

一个用于基于云原神的每日领取月卡脚本

## 前置条件

确保你已安装 Node.js 22 或更高版本。如果尚未安装，请访问 [Node.js 官网](https://nodejs.org) 下载并安装最新版本的 Node.js。

确保安装 Puppeteer 所需的 Chrome 浏览器，可以通过以下命令来安装它：

```bash
npx puppeteer browsers install
```

## 安装

```bash
npm install genshin-freedom -g
```

## 使用

```bash
genshin-start
```

初次使用会弹出云原神网页，登录完成后，后续可正常使用
